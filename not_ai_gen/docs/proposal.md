# Project Proposal

## Title
Multi-Agent Fictional Personalities

### Idea in one sentence
I build a system that transforms fictional characters into conversational agents and measures whether their responses are recognizable and consistent with the original character.

### Research question
Can LLMs, conditioned on persona profiles extracted from fictional character dialogues, simulate characters that are distinguishable from one another in a multi-agent chat?

### Main hypothesis
The responses generated by the character agents will be attributed to the correct character by human evaluators with accuracy above chance.

### Project profile
Mixed: part of the work involves building the system, and part involves designing and conducting an evaluation.

#### Initial scope
- Number of characters: 4–6 for the Minimum Viable Build
- Data source: dialogues / transcripts / structured descriptions
- Model: a single, configurable model
- Persona extraction: one initial method
- UI: a simple interface to simulate chats and inspect profiles
- Evaluation: character identification task

#### Out of scope
- Fine-tuning
- Complex persistent memory
- Comparing multiple models within the same experiment
- Overly elaborate UI
- Psychological study of the characters
