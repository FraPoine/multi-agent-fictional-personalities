# Roadmap

The target for the first usable release is August 7, 2026. The final course deadline is in September; no exact date is currently documented.

## Sprint 1

Architecture, specification, evaluation design, repository structure, and initial corpus work.

## Sprint 2

Minimal corpus-to-persona-to-response pipeline for Sherlock Holmes and Hercule Poirot.

## Sprint 3

Multi-agent round-robin conversation with transcript and JSONL logging.

## Sprint 4

Evaluation trial generation, basic analysis, smoke test, reproducibility, and first usable release by August 7.

## Presentation sprint

Full four-character experiment, human evaluation, confidence intervals, report, limitations, and final presentation.
